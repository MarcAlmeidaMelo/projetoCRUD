 Projeto CRUD
 
O Projeto CRUD é uma aplicação desenvolvida com o objetivo de demonstrar, de forma prática, a implementação das operações fundamentais de manipulação de dados: Create, Read, Update e Delete (CRUD).

A aplicação permite o gerenciamento de registros (como usuários), simulando um sistema básico de persistência de dados, sendo uma base essencial para sistemas mais complexos.

Objetivo


Consolidar conceitos fundamentais de desenvolvimento web
Aplicar operações básicas de manipulação de dados (CRUD)
Trabalhar a integração entre interface e lógica de negócio
Servir como base para evolução para sistemas mais robustos
  
  Funcionalidades:
   Criação de novos registros;
   Listagem de dados cadastrados;
   Atualização de informações;
   Remoção de registros;
   Interação dinâmica com os dados;
   Tecnologias Utilizadas;


Linguagem: C#;
Banco de Dados: Oracle;
HTML5 & CSS3;
Git & GitHub;
